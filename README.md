# Weather App - Real-Time Weather Application

A full-stack weather application that provides real-time weather data, city search with autocomplete, search history tracking, and current location weather detection. Built with the MERN stack (MongoDB, Express.js, React, Node.js).


## Features

- **City Search** - Search for weather in any city worldwide
- **Autocomplete Suggestions** - Real-time city name suggestions as you type
- **Current Location** - Get weather for your exact location (GPS)
- **Detailed Weather Info** - Temperature, humidity, wind speed, sunrise/sunset
- **Search History** - Track your recent searches (last 10)
- **History Management** - Delete individual or all search history
- **Glassmorphism UI** - Modern, beautiful, responsive design
- **Fully Responsive** - Works perfectly on desktop, tablet, and mobile
- **Dark Theme** - Eye-friendly dark mode design

## Technology Stack

### Frontend
- **React 18** - UI Library
- **Tailwind CSS** - Utility-first CSS framework
- **Material-UI (MUI)** - Component library and icons
- **Axios** - HTTP client for API requests
- **React Hot Toast** - Toast notifications

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB ODM
- **Axios** - External API calls

### APIs
- **OpenWeatherMap API** - Weather data provider

## Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v14 or higher)
- **npm** or **yarn** package manager
- **MongoDB** (local installation or MongoDB Atlas account)
- **OpenWeatherMap API Key** ([Get free API key](https://openweathermap.org/api))
